Dashboard & Grafik

Lihat saldo, total pemasukan/pengeluaran, dan grafik per kategori yang langsung update otomatis.

Laporan Bulanan

Rekap otomatis tiap akhir bulan: ke mana uangmu pergi dan kategori paling boros.

PERTANYAAN UMUM

Masih ragu?

Apakah ini aplikasi yang harus di-install?

Tidak. SanguKu versi web cukup dibuka di browser HP atau laptop — tinggal klik. Kamu juga dapat template Excel untuk yang lebih suka spreadsheet.

Data keuangan saya aman?

Sangat aman. Data tersimpan di perangkatmu sendiri, bukan di server kami. Tidak perlu daftar akun atau berikan info pribadi.

Saya gaptek, apakah bisa pakai?

Bisa banget. SanguKu dirancang sesederhana mungkin — input transaksi semudah mengirim chat. Ada panduan langkah demi langkah juga.

Bagaimana cara dapat produknya setelah bayar?

Setelah pembayaran, kamu langsung menerima link akses aplikasi dan file template via email/WhatsApp. Bisa langsung dipakai hari itu juga.

Apakah ada biaya bulanan?

Tidak ada. Cukup bayar sekali dan pakai selamanya, termasuk semua update di masa depan.

Apakah aplikasi & template Excel saling terhubung?

Keduanya alat terpisah — kamu bebas pakai salah satu atau dua-duanya. Data di aplikasi tidak otomatis masuk ke Excel. Tapi di aplikasi ada menu Export CSV yang bisa kamu buka/tempel ke Excel kapan pun kalau ingin memindahkan data.
